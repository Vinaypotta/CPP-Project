# CPP-Project
This project aims to provide a comprehensive codebase that replicates real-world software development scenarios.
<h3> Rules </h3>
    <li>Each project must have a common base class that all other classes inherit from.</li>
    <li>All common base classes must inherit from Base.h.</li>
    <li>Common base classes must contain project metadata including project details, class information, functionality, and implemented topics.</li>
    <li>Common base classes should be named after their respective projects.</li>
    <li>Create a new branch for each project and merge it into main upon completion.</li>
    <li>Update the Base class documentation when merging projects.</li>
    <li>Every project should be created in a folder.</li>
</ul>
